# Frontend Mentor - Clipboard Landing Page

## 📌 Overview

This is a solution to the Clipboard Landing Page challenge from Frontend Mentor.  
The goal of this project is to build a responsive landing page that matches the provided design as closely as possible.

- View the optimal layout depending on their device's screen size
- See hover states for all interactive elements
